# Themes

Petitorium supports a wide range of syntax highlighting themes. Many of these provide "Unified Theming," which styles the entire application UI to match the code highlights.

## Listing Available Themes

To see all available themes on your system, run:

```bash
petitorium themes
```

## Popular Unified Themes

Petitorium includes high-quality themes out of the box:

- **Nord** (Default)
- **Dracula**
- **One Dark**
- **Monokai**
- **Gruvbox**
- **Catppuccin Mocha**
- **Solarized Dark**
- **Tokyonight Night**

## Switching Themes

You can change the application theme instantly from the CLI:

```bash
petitorium themes switch dracula
```

:::info
The new theme will be applied the next time you start Petitorium.
:::

## Customization

Themes are defined in your configuration file. You can adjust colors and styles to perfectly match your terminal emulator's aesthetic.
